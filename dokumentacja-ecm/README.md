# ECM Digital - Dokumentacja

## Wprowadzenie

ECM Digital to agencja interaktywna specjalizująca się w tworzeniu stron internetowych, sklepów internetowych na platformie Shopify, prototypów MVP aplikacji oraz przeprowadzaniu audytów UX. Niniejsza dokumentacja służy jako kompleksowe źródło informacji o ofercie, procesach i kompetencjach agencji.
